# Veil Of The Sword
A simple text-based game, made with python to use various concepts, like Tuples, Classes, Decorators, and more. Inspired by the [Swords And Sandals](https://swordsandsandals.fandom.com/wiki/Swords_and_Sandals_Wiki) franchise.

## Installation
- Clone the repo and enter the dir  
- Run `python3 src/main.py`
